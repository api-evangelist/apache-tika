# Apache Tika (apache-tika)
Apache Tika is a toolkit for detecting and extracting metadata and structured text content from various documents using existing parser libraries. It supports over 1000 file formats including PDF, Word, Excel, HTML, and multimedia formats.

## APIs

### Apache Tika REST API
Tika provides a REST API (tika-server) for content detection and extraction, a Java API for parsing documents, and a comprehensive parser API supporting over 1000 file formats including PDF, Office documents, images, audio, and video.
